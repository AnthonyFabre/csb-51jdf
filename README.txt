Commande du jeu :

right : aller a droite
left : aller a gauche
up : sauter
Z : tirer 
A : téléporter
space : Franchir une porte ou prendre le chien

-  Nous aurions aimé ajouter un bouton pour rejouer après être mort ou après avoir gagné.
-  Ajoutez peut-être des sons de saut, de porte, de chien ou de singe pour apporter plus de dynamisme.
-  Pour plus de difficulté, placer des pièces pour que le joueur les ramasse toutes. Si les pièces ne sont pas collectées, la porte permettant de passer le niveau ne s'affiche pas.
